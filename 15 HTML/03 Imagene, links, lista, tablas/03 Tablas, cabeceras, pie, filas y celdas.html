Como construir tablas utilizamos la etiqueta de table

Las tablas se construyen de 3 secciones cabecera , el contenido y la parte inferior

Para definir la caberzera utilizamos la etiqueta de thead, par indicar la cantidad de filas utilizamos la etiqueta tr y para el continido se coloca th

para ver el contenido utilizaremos la etiqueta tbody para mostar el contenido de esta se usa la tr nuevamente y para mostrar el contenido se utiliza td

Para realizar el pie de la tabla se utiliza la etiqueta tfoot dentro de esta se utiliza la etiqueta tr y dento de esta td
