Si no tengo definido algun parametro en concreto uso directo la etiqueta a
a{
color: hotpink;
}
Si solo quieres modificar el parametro de link de la etiqueta a se realiza de la siguiente manera.
a:link{
color: pink;
}
Si quieres cambiar el estilo de cuando el link fue visitado se realiza de la siguiente manera
a:visited{
color: hotpink;
}
Para cambiar el color cuando posamos el cursor sobre el link se modifica el parametro 'hover'
a:hover{
color: gray;
}
Para cambiar el color cuando se pincha el link se usa el parametro active de la etiqueta a
a:active{
color: black;
}
