En HTML todas las etiquetas que nosotros escribimos o las que nosotros vamos a redactar aquí dentro de nuestros archivos punto HTML, son considerados cajas.

Hola Mundo

Cuando uno aprienta el inspector de elementos y coloca el mouse sobre el div nos muestra el espacio de caja que este esta utilizando en la pagina.
Si vemos la de body nos muetra que esta toma la caja del div y hasta esta donde llega el contenido de esta etiqueta a su vez nos muestra el margen que esta etiqueta nos da de un elemento del otro.
Ahora al div le agregaremos la clase 'caja' que creamos en el archivo css e inspeccionamos como se ve por el inspector de elementos.
