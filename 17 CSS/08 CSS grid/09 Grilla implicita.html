La grilla implícita es todo aquel contenido que se encuentre dentro de este grid que no haya sido definido dentro de nuestros grid, grid-template-rows, grid-template-columns o también dentro de nuestro grid-template-areas. Supongamos que tenemos solo 3 filas agregadas en el grid y agregamos mas contenido seguiran apareciendo pero no directa por asi decirlo, si a la propiedad de grid en el css le agregamos el parametro grid-auto-rows este nos ayudadra a colocar de manera automatica el tamaño a todo elemento que este fuera de los parametros señalados con anterioridad de la siguiente manera: 
            CSS:
            .grid{
                display: grid;
                grid-template-areas: "header herader" "content sidebar" "footer footer";
                grid-template-rows: 150px 1fr 100px;
                grid-template-columns: 1fr 200px;
                grid-auto-rows: 140px;
            }
